Fresh Windows Setup:

Install winget
	Invoke-WebRequest -Uri https://aka.ms/getwinget -OutFile $ENV:TEMP\winget.msixbundle; 
	Add-AppXPackage $ENV:TEMP\winget.msixbundle;

	winget search terminal # To list packages
	winget install Microsoft.WindowsTerminal


Bash V1
	winget install Git.Git

	Follow the install instructions and when that is done, go to Settings in the terminal and do the following:
	Select Add a new profile
	Create a New empty profile
	Name the profile Git Bash
	Set the Command line to %ProgramFiles%\Git\bin\bash.exe
	Set the Icon to %ProgramFiles%\Git\mingw64\share\git\git-for-windows.ico
	Press Save to save the profile.
